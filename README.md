# photo-gallery-administrator
手芸作家の作品ギャラリーサイトの投稿管理用アプリです。 

ギャラリーサイトURL  
https://hiiragi-ya.net/

この管理アプリの機能  
・作品の新規追加  
・ギャラリーサイト側での公開設定   
・投稿の削除  
・投稿内容の編集  

・ステージング環境URL
http://admin-staging.uh-oh.jp/controller/IndexController.php    
※検証環境ですので自由に操作していただいて問題ありません。
